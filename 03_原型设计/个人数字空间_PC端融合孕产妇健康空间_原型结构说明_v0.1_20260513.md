## 四、孕产妇健康空间详情页

### 4.1 页面定位

承载当前患者在"孕产妇健康空间"下的孕产期全量健康数据。独立详情页，支持产前/产后两种状态差异化展示。

### 4.2 页面整体结构

采用：**2×2 四模块卡片网格**

```text
┌───────────────────────────────────────────────────────────┐
│ 页面头部                                                    │
│ 第1层：返回入口 | 孕产妇健康空间                              │
│ 第2层：姓名 性别 年龄 证件号 | 数据截止时间                   │
│ 第3层：[孕中期] 第24+3周  预产期：2026-09-15（距今127天）     │
│        ●━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━○   │
├───────────────────────────────────────────────────────────┤
│ ┌─────────────────────────┐  ┌─────────────────────────┐ │
│ │ 卡片A：孕产健康视图      │  │ 卡片B：孕期核心指标      │ │
│ │  子菜单/搜索/列表        │  │  子菜单/统计/趋势图      │ │
│ └─────────────────────────┘  └─────────────────────────┘ │
│ ┌─────────────────────────┐  ┌─────────────────────────┐ │
│ │ 卡片C：孕期风险预警      │  │ 卡片D：产检记录          │ │
│ │  风险指标/等级/干预建议   │  │  时间线/关键检查项目     │ │
│ └─────────────────────────┘  └─────────────────────────┘ │
└───────────────────────────────────────────────────────────┘
```

### 4.3 页面头部

#### 结构

页面头部分三层，全部固定不随页面滚动：

| 层级 | 内容 | 原型表现建议 |
|------|------|------------|
| 第1层 | 返回按钮 + "孕产妇健康空间"标题 | 左对齐，返回按钮用箭头图标 |
| 第2层 | 姓名、性别、年龄、证件号、数据截止时间 | 横向信息条，与通用人群详情页头部一致 |
| 第3层 | 孕周进度区（孕产妇专属） | 见下方详细说明 |

#### 第3层：孕周进度区

**产前状态**：
- 左侧：孕期阶段标签（孕早期/孕中期/孕晚期）
- 中间：当前孕周（格式"第N+N周"）
- 右侧：预产期 + 距今倒计时
- 下方：进度条，标注四阶段分界点

**产后状态（产褥期）**：
- 左侧：产褥期标签
- 中间：分娩日期
- 右侧：产后第N天 + 距空间关闭剩余天数
- 下方：产后进度条（分娩→产后42天）

#### 原型重点体现
- 产前/产后两种状态需各出一张图
- 进度条当前位置用三角标记

### 4.4 卡片A：孕产健康视图

#### 卡片内部结构

```text
┌─────────────────────────────────────────┐
│ 卡片标题：孕产健康视图                    │
├─────────────────────────────────────────┤
│ 子菜单：全部|孕早期|孕中期|孕晚期|分娩期|产褥期 │
├─────────────────────────────────────────┤
│ 搜索筛选区：关键词输入 + 时间范围选择     │
├─────────────────────────────────────────┤
│ 列表区（卡片内独立滚动）                  │
│  ┌─ 记录1：诊断名称 | 类型 | 机构 | 时间 │
│  ├─ 记录2：...                          │
│  └─ 记录N：...                          │
└─────────────────────────────────────────┘
```

#### 交互要点
- 子菜单每项后展示数据量角标
- 默认选中"当前所处孕期阶段"
- 点击列表项 → 弹出二级弹窗（诊疗详情/住院详情/随访详情）
- 搜索筛选区随卡片内容滚动

#### 弹窗交互（与通用人群一致）
- 二级弹窗：居中大弹窗（宽度80%视口），展示详情字段
- 三级弹窗：覆盖二级弹窗，展示检查/检验/处方/手术明细
- 弹窗头部：左侧"返回"按钮 + 右侧"关闭"按钮

### 4.5 卡片B：孕期核心指标

#### 卡片内部结构

```text
┌─────────────────────────────────────────┐
│ 卡片标题：孕期核心指标                    │
├─────────────────────────────────────────┤
│ 子菜单：血压 | 血糖 | 体重               │
├─────────────────────────────────────────┤
│ 统计区：孕产期间检查N次 | 异常N次         │
├─────────────────────────────────────────┤
│ 趋势图区（卡片内独立滚动）                │
│  [趋势图：横坐标日期，纵坐标数值]         │
│  [体重趋势图叠加推荐区间参考线]           │
├─────────────────────────────────────────┤
│ 缩放进度条占位                            │
└─────────────────────────────────────────┘
```

#### 交互要点
- 默认选中"血压"
- 点击趋势图数据点 → 弹出溯源弹窗
- 体重趋势图需展示推荐增长区间（上下限参考线）
- 体重子菜单下方展示文案提示区

#### 与通用人群体征数据的差异（原型需体现）
- 子菜单"BMI"替换为"体重"
- 统计口径为"本次孕产期间"（非历年/近一年）
- 体重趋势图有参考线

```text
┌─────────────────────────────────────────┐
│ 卡片标题：孕期风险预警                    │
├─────────────────────────────────────────┤
│ 风险指标列表（卡片内独立滚动）            │
```

#### 交互要点
- 按风险等级降序排列（高 > 中 > 低）
- 风险等级用色块/图标区分（高=红、中=橙、低=黄）
- 无命中风险时显示"暂无风险预警"空状态
- 无下钻交互

## 五、产后状态特殊展示

产褥期状态下，原型需体现以下差异：

| 区域 | 产前 | 产后（产褥期） |
|------|------|-------------|
| 头部第3层 | 孕期进度条 | 产后进度条（分娩→42天） |
| 孕产健康视图默认Tab | 当前孕期阶段 | 产褥期 |
| 体重趋势图参考线 | 孕期推荐增长区间 | 产后恢复推荐区间 |
| 风险预警指标集 | 孕期风险指标 | 产后风险指标 |

## 六、弹窗规范

### 6.1 二级弹窗（诊疗详情/住院详情/随访详情）

- 尺寸：居中，宽度 80% 视口，高度自适应（最大 80% 视口高度）
- 头部：左侧标题（如"门诊详情"）+ 右侧"关闭"按钮
- 内容区：支持内部滚动
- 背景：半透明遮罩

### 6.2 三级弹窗（检查/检验/处方/手术明细）

- 尺寸：与二级弹窗一致
- 层级：覆盖二级弹窗（二级弹窗保留但不可交互）
- 头部：左侧"返回"按钮 + 标题 + 右侧"关闭"按钮
- "返回"：关闭三级弹窗，回到二级弹窗（保留滚动位置）
- "关闭"：关闭所有弹窗，回到卡片列表

### 6.3 溯源弹窗

- 尺寸：小弹窗，靠近数据点位置
- 内容：溯源字段（与通用人群一致）
- 关闭：点击弹窗外区域或关闭按钮
