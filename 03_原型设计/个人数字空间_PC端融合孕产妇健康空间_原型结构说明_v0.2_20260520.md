# 个人数字空间_PC端融合孕产妇健康空间_原型结构说明_v0.2_20260520

## 一、文档定位

本文档配套 `个人数字空间_PC端融合孕产妇健康空间_原型_v0.2_20260520.html`，基于需求文档 `个人数字空间_PC端融合孕产妇健康空间_需求文档_v0.3_20260513.md` 对原型结构进行同步说明。

v0.2 的目标是修正 v0.1 原型与 v0.3 需求之间的差异：详情页恢复为 5 模块结构，下钻统一为 Modal，并按医疗后台原型规范收敛视觉和交互。

---

## 二、页面范围

本次原型包含 2 类视图：

1. 个人数字空间主界面入口态
   - 保持既有主界面结构，不重新设计主界面。
   - 孕产妇标签升级为正常可点击入口。
   - 其他专项人群仍按后续开放入口处理。

2. 孕产妇健康空间详情页
   - 支持产前、产后两种演示状态。
   - 支持孕产记录筛选、指标切换、风险预警、产检时间线、AI 摘要展示。
   - 支持二级详情 Modal 和三级明细 Modal。

---

## 三、详情页结构

详情页采用 `页面头部 + 2×2 卡片网格 + 底部全宽卡片`。

```text
孕产妇健康空间详情页
  ├─ 固定页头
  │  ├─ 返回入口 / 页面标题
  │  ├─ 患者基础信息 / 数据截止时间
  │  └─ 孕周或产后进度区
  ├─ 2×2 模块网格
  │  ├─ 孕产健康视图
  │  ├─ 孕期核心指标
  │  ├─ 孕期风险预警
  │  └─ 产检记录
  └─ 底部全宽模块
     └─ 孕期AI摘要与个性化建议
```

### 3.1 页面头部

- 第 1 层：返回个人数字空间、面包屑、当前标题。
- 第 2 层：姓名、性别、年龄、证件号、数据截止时间。
- 第 3 层：孕周进度区或产后 42 天进度区。
- 原型状态切换保留在第 3 层右侧，仅用于评审演示产前/产后差异。
- 页头不放说明段落、英文副标题或额外提示文案。

### 3.2 孕产健康视图

- 默认选中当前所处阶段：产前为孕中期，产后为产褥期。
- 子菜单包含：孕中期、全部、孕早期、孕晚期、分娩期、产褥期。
- 支持关键词、开始日期、结束日期筛选。
- 点击记录进入二级详情 Modal。

### 3.3 孕期核心指标

- 子菜单包含：血压、血糖、体重。
- 统计口径为本次孕产期间。
- 趋势图使用轻量图表样式：浅灰虚线网格、细蓝折线、空心点。
- 体重指标保留推荐区间参考线。
- 点击趋势点进入指标溯源 Modal。

### 3.4 孕期风险预警

- 风险按等级降序排列：高风险、中风险、低风险。
- 每条风险包含风险名称、评估依据、干预建议、监测指标。
- 无下钻交互。

### 3.5 产检记录

- 使用时间线展示历次产检。
- 每条记录包含产检时间、孕周、机构、关键检查项和结论。
- 无下钻交互。

### 3.6 孕期AI摘要与个性化建议

- 底部全宽展示。
- 内容分为当前孕产状态、风险提醒、随访建议、个性化建议。
- 内容为原型假数据，不新增业务规则。

---

## 四、弹窗规范

### 4.1 二级详情 Modal

- 居中弹窗，宽度为 80% 视口，最大高度约 82% 视口。
- 右上角使用线性 SVG 关闭按钮。
- 记录基础信息按字段卡片分组展示。
- 有检查、检验、处方、手术数据时，在二级 Modal 内提供三级明细入口。

### 4.2 三级明细 Modal

- 三级 Modal 覆盖二级 Modal，二级 Modal 保留在下层。
- 左侧提供返回按钮，右侧提供关闭全部弹窗按钮。
- 检验类以表格展示；检查、处方、手术类以字段分组展示。

## 五、产前/产后差异

| 区域 | 产前 | 产后 |
|------|------|------|
| 头部进度区 | 孕中期、第 24+3 周、预产期倒计时 | 产褥期、分娩日期、产后第 42 天 |
| 孕产健康视图默认 Tab | 孕中期 | 产褥期 |
| 核心指标数据 | 孕期血压、血糖、体重趋势 | 产后恢复期血压、血糖、体重趋势 |
| 风险预警 | GDM、贫血、胎儿发育等孕期风险 | 糖耐量复查、产后抑郁、伤口愈合等产后风险 |
| AI 摘要 | 孕期管理建议 | 产后恢复与复查建议 |

---

## 六、样式约束

- 白色卡片承载业务模块，页面背景使用浅灰。
- 卡片圆角统一为 8px，输入框和按钮高度统一为 32px。
- 子菜单使用轻量分段矩形标签，激活态为品牌蓝实底白字。
- 图标统一使用线性 SVG，不使用 emoji 或单字符伪图标。
- 弹窗关闭按钮使用 SVG 叉号。
- 删除无业务必要的说明性文案，页面信息层级聚焦当前功能。
