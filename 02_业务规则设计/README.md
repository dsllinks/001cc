# 业务规则目录

## 目录说明

存放各类业务规则定义文档，回答"**怎么判断**"——标签命中条件、指标计算公式、字段来源与取值范围。

这类文档由产品经理产出，用于研发团队统一产出全量标签和指标数据，是产品迭代的规则依据。

> 与需求文档的区别：需求文档描述功能场景和交互逻辑（做什么），业务规则文档定义具体的判断逻辑和计算规则（怎么判断）。
