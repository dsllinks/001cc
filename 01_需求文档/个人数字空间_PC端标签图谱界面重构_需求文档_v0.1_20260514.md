## 1. 背景与目标

### 1.1 背景

当前 PC 端个人数字空间主界面的核心功能为"个人健康标签图谱"，采用放射分层图形态（中心患者头像 → 一级类别 → 二级具体标签），支持按标签类别筛选、查看可展开/不可展开/可下钻标签、展示管理标签集可见范围。

随着 V1.1 版本标签体系拓展（新增机构标签、医生标签）和重点人群覆盖扩大，现有图谱存在以下问题：

1. **类别布局密集时严重重叠**：标签数量增多后，类别名称与二级标签互相穿插，文字方向各异（横/斜/竖），可读性大幅下降
2. **节点视觉单一**：仅用填充色区分类别，无风险等级、无时效、无数据来源视觉差异
3. **类别间无逻辑分组**：9-10个类别等权环绕，无主次区分
4. **二级标签文字方向问题**：沿圆周切线方向旋转，斜置时阅读吃力
5. **类别名识别困难**：类别名压在中圈连线上，字号小
6. **无层级折叠/聚焦能力**：不能专注查看单个类别，必须看全部

### 1.2 建设目标

- 优化放射分层图的结构布局，解决标签密集时的可读性问题
- 增强节点视觉差异化，支持风险等级快速识别
- 新增类别折叠/聚焦能力，支持医生按需聚焦单类别
- 补齐搜索定位和悬停高亮交互，提升大量标签场景下的操作效率
- 完善三级标签下钻路径和溯源面板内容
- 满足合规要求，增加数据脱敏标识和数据时效声明

### 1.3 非目标

- 不改变主界面三区布局（个人信息区 / 标签及人群统计卡 / 标签图谱区）
- 不改造二级详情页（通用人群/孕产妇/慢病健康空间详情）
- 不新增功能域（不引入 AI 摘要、健康建议等新模块）
- 不重做标签生成能力（规则/KIE/大模型）
- 不改造标签管理流程（创建/审核/发布/下线）

- 左侧图例与筛选区
- 右侧溯源信息面板

| 模块 | 已有能力 | 本期重构要求 |
|------|---------|------------|

- 数据来源角标图标
- 时效性透明度编码
- 生成方式标识（tooltip/面板）
- 可追溯入口（跳转规则/模型说明）
- 置信度提示
- 审计日志入口

```
  │
  ├─ 悬停二级/三级节点 → 高亮该节点+所属类别+连线，其他淡化
  │
  └─ 搜索框输入 → 匹配节点高亮+自动平移定位
```

## 4. 主界面变更说明

### 4.1 整体说明

本次重构仅涉及主界面第三区"个人健康标签图谱区"，不改动第一区（个人信息）和第二区（标签及人群统计卡）。

图谱区的整体容器尺寸和位置不变，内部结构重新组织。

### 4.2 图谱区布局变更

| 维度 | 旧方案 | 新方案 |
|------|--------|--------|
| 左侧区域 | 图例+管理标签集下拉 | 取消独立左侧区域，筛选移至顶部工具栏 |
| 顶部 | 无 | 新增工具栏（筛选器+搜索框）+ 合规标识 |
| 图谱主区域 | 占据右侧全部空间 | 占据工具栏下方全部空间 |
| 右侧 | 溯源面板（点击时弹出） | 保留，内容扩展 |

### 4.3 合规标识展示规则

- **数据脱敏标识**：图谱区右上角固定展示徽章"已脱敏展示"，始终可见
- **数据时效声明**：图谱区顶部展示"数据截止：YYYY-MM-DD"，取值与主界面患者头部一致

## 5. 功能模块详细需求

### 5.1 放射分层图布局优化

#### 5.1.1 功能说明

在保留"中心患者→一级类别→二级标签"三层放射结构的基础上，优化类别名展示、文字方向和扇区分配规则。

#### 5.1.2 类别名视觉强化

- 类别名（如"疾病风险"、"用药"等）独立成弧形章节标签
- 弧形标签带浅色背景条，与二级节点视觉分离
- 字号不小于14px，颜色与所属类别色板一致但加深
- 弧形标签位于一级层与二级层之间，不压在连线上

#### 5.1.3 二级标签文字水平化

- 所有二级标签节点内的文字始终保持水平方向
- 节点在圆周上的位置不变，仅调整文字朝向
- 文字超长时截断显示，悬停 tooltip 展示全文

#### 5.1.4 扇区角度动态分配

- 各类别占据的扇区角度按该类别下二级标签数量动态计算
- 计算公式：类别扇区角度 = (该类别标签数 / 总标签数) × 360°
- 最小扇区角度不低于 20°（避免标签过少的类别被压缩到不可见）
- 最大扇区角度不超过 150°（避免单类别独占）

#### 5.1.5 缩放与平移

- 滚轮缩放：支持 50%~200% 范围
- 鼠标拖拽：整图平移（节点不可单独拖拽，保持布局稳定）
- 右上角操作按钮：+（放大）/ -（缩小）/ "自适应"（一键回到100%居中）
- 缩放时节点文字保持可读（低于80%时文字隐藏，仅显示色块）

### 5.2 节点视觉差异化

#### 5.2.1 功能说明

在保留类别填充色和可下钻角标的基础上，新增风险等级边框编码，使医生能快速识别高危标签。

#### 5.2.2 风险等级边框

| 风险等级 | 边框样式 | 说明 |
|---------|---------|------|
| 高 | 红色粗描边（3px，#EF4444） | 需要立即关注的高危标签 |
| 中 | 橙色中等描边（2px，#F97316） | 需要持续观察的中危标签 |
| 低 | 灰色细描边（1px，#9CA3AF） | 低风险标签 |
| 无 | 无描边 | 非风险类标签（如基础信息） |

#### 5.2.3 保留的视觉编码

- **类别填充色**：沿用现有10类色板（用药/疾病风险/九大系统风险/检查风险/检验风险/手术/基础信息/十大人体器官风险/模型健康/健康空间）
- **可下钻角标**：节点右下角 ◎ 图标，表示可展开三级标签

#### 5.2.4 三级标签节点视觉

- 三级节点尺寸比二级小一号（二级直径40px，三级直径28px）
- 颜色继承所属类别色
- 保留风险等级边框和可下钻角标
- 文字水平，超长截断

### 5.3 类别折叠/聚焦

#### 5.3.1 功能说明

支持医生点击类别名进入聚焦模式，专注查看单个类别的全部标签。

#### 5.3.2 交互规则

| 操作 | 效果 |
|------|------|
| 点击类别弧形标签 | 该扇区放大占据60%圆，其他类别压缩为外圈小节点（仅显示类别名，不显示二级标签） |
| 再次点击同一类别 | 恢复全景模式 |
| 点击中心头像 | 恢复全景模式 |
| 聚焦模式下点击其他类别 | 切换聚焦到新类别（无需先回全景） |

#### 5.3.3 聚焦模式下的展示规则

- 聚焦类别的二级标签全部展示（不受 Top-N 限制）
- 被压缩的类别仅显示类别名+标签总数（如"用药 ×12"）
- 聚焦模式下仍可点击二级标签进入三级展开

### 5.4 悬停高亮与 Tooltip

#### 5.4.1 功能说明

鼠标悬停节点时高亮关联元素并显示摘要信息，帮助医生在密集图谱中快速锁定目标。

#### 5.4.2 悬停高亮规则

| 悬停对象 | 高亮效果 | 淡化效果 |
|---------|---------|---------|
| 二级标签节点 | 该节点放大1.2倍 + 所属类别弧形标签高亮 + 连接线加粗 | 其他类别所有节点淡化至30%透明度 |
| 三级标签节点 | 该节点放大1.2倍 + 所属二级节点高亮 | 其他三级节点淡化至30% |
| 类别弧形标签 | 该类别所有二级节点高亮 | 其他类别淡化至30% |

#### 5.4.3 Tooltip 内容

悬停节点 200ms 后显示 tooltip，内容：
- 标签名（全文，解决截断问题）
- 风险等级（高/中/低/无）
- 生成时间（YYYY-MM-DD）

Tooltip 位置：节点上方居中，避免遮挡其他节点。

### 5.5 搜索定位

#### 5.5.1 功能说明

支持按标签名关键词搜索，快速定位目标节点。

#### 5.5.2 搜索交互

- 搜索框位于图谱区右上角（缩放按钮左侧）
- 输入关键词后实时匹配二级和三级标签名（模糊匹配）
- 匹配结果 >1 时，搜索框下方展示下拉列表（最多显示10条）
- 选中某条 → 图谱自动平移使该节点居中 + 节点高亮闪烁2次
- 匹配结果 =1 时，直接定位
- 匹配结果 =0 时，搜索框下方提示"未找到匹配标签"

#### 5.5.3 搜索范围

- 默认搜索当前筛选条件下可见的标签
- 如搜索无结果，提示"当前筛选条件下未找到，是否搜索全部标签？"

### 5.6 三级标签下钻

#### 5.6.1 功能说明

点击二级标签后，该节点居中放大并向外辐射展开三级标签，同时自动打开溯源面板。

#### 5.6.2 展开规则

| 规则 | 说明 |
|------|------|
| 展开方向 | 围绕该二级节点辐射展开，180°扇形朝外（避免与其他类别交错） |
| 数量上限 | 单二级展开三级 ≤12 个 |
| 超限处理 | 超过12个时，按风险等级降序+时间倒序取前12个，末尾显示"+N"汇聚节点 |
| 动画 | 展开动画300ms，节点依次出现（间隔50ms） |

#### 5.6.3 三级交互

| 操作 | 效果 |
|------|------|
| 点击三级标签 | 溯源面板内容更新为该三级标签的命中证据 |
| 点击其他二级标签 | 收起当前三级，展开新二级的三级（直接切换，无需先回全景） |
| 点击中心头像 / Esc | 收起三级，回到全景模式 |
| 点击"+N"汇聚节点 | 展开剩余三级标签（替换汇聚节点） |

#### 5.6.4 不支持第四级

三级是图谱内的最深层。如需更细数据，通过溯源面板的命中证据区块查看原始事实记录。

### 5.7 溯源信息面板

#### 5.7.1 功能说明

点击二级/三级标签时，右侧展开溯源信息面板，展示该标签的基本信息和命中证据。

#### 5.7.2 面板结构

```
┌─ 溯源信息 ──────────────────────────┐
│ ◆ 基本信息                           │
│   标签名：{标签名全文}                │
│   所属类别：{类别名}                  │
│   风险等级：{高/中/低/无}             │
│   生成时间：{YYYY-MM-DD HH:mm}       │
│   数据截止：{YYYY-MM-DD}             │
├──────────────────────────────────────┤
│ ◆ 命中证据                           │
│   首次出现：{YYYY-MM-DD}             │
│   医疗机构：{机构名称}                │
│   科室：{科室名称}                    │
│   诊断/检查/检验/用药原文：{原文}     │
│   关联事实数：{N}条                   │
└──────────────────────────────────────┘
```

#### 5.7.3 面板交互

| 操作 | 效果 |
|------|------|
| 点击新节点 | 面板内容更新为新节点信息（面板保持打开） |
| 点击面板关闭按钮 / Esc | 面板收起，图谱保留选中状态 |
| 面板打开时点击中心头像 | 面板收起 + 图谱回到全景 |

#### 5.7.4 空数据规则

- 命中证据区块某字段无数据时，显示"暂无数据"（灰色文字）
- 关联事实数为0时，命中证据区块整体显示"该标签暂无关联事实记录"

### 5.8 筛选能力

#### 5.8.1 功能说明

在图谱区顶部工具栏提供多维筛选，替代原左侧"管理标签集可见"勾选。

#### 5.8.2 筛选项

| 筛选维度 | 控件形式 | 取值 | 默认值 |
|---------|---------|------|--------|
| 标签类别 | 多选下拉 | 10类（用药/疾病风险/九大系统风险/检查风险/检验风险/手术/基础信息/十大人体器官风险/模型健康/健康空间） | 全选 |
| 时间窗 | 单选下拉 | 近30天 / 近90天 / 近1年 / 全部 | 全部 |

#### 5.8.3 筛选规则

- 两个筛选项之间为 AND 关系
- 筛选结果为空时，图谱区显示空态："当前筛选条件无匹配标签，请调整筛选条件"
- 筛选变更后，图谱实时刷新（无需点击"查询"按钮）
- 筛选状态在页面刷新后重置为默认值

#### 5.8.4 图例说明

原左侧图例（可展开/不可展开/可下钻）改为图谱区右下角浮窗：
- 默认折叠，仅显示"图例"文字按钮
- 点击展开显示三种节点类型说明
- 新增风险等级边框图例说明

## 7. 端适配规则

### 7.1 布局适配原则

- 图谱区占据主界面第三区全部空间（宽度自适应）
- 最小支持宽度：1280px（低于此宽度不做特殊适配）
- 溯源面板宽度固定320px，从右侧推入（不覆盖图谱区，图谱区宽度相应缩减）
- 顶部工具栏横向排列，窄屏时筛选项自动换行

### 7.2 业务规则复用清单

| 规则类型 | 复用内容 |
|---------|---------|
| 标签类别定义 | 沿用标签管理平台现有10类分类 |
| 风险等级定义 | 沿用标签管理平台标签属性中的风险等级字段 |
| 标签可见范围 | 沿用"管理标签集可见"的权限控制逻辑 |
| 数据截止时间 | 沿用主界面患者头部的数据截止时间取值逻辑 |
| 溯源信息字段 | 沿用现有溯源面板的数据查询接口 |

### 7.3 默认展示规则

| 模块 | 默认状态 |

## 9. 验收标准

### 9.1 放射分层图布局

- [ ] 类别名独立成弧形章节标签，带浅色背景条，不压在连线上
- [ ] 所有二级标签文字保持水平方向，不随圆周旋转
- [ ] 扇区角度按标签数量动态分配，最小20°，最大150°
- [ ] 支持滚轮缩放50%~200%，鼠标拖拽平移
- [ ] 右上角+/-/"自适应"按钮功能正常
- [ ] 缩放低于80%时文字隐藏，仅显示色块

### 9.2 节点视觉

- [ ] 高风险标签显示红色粗描边（3px）
- [ ] 中风险标签显示橙色中等描边（2px）
- [ ] 低风险标签显示灰色细描边（1px）
- [ ] 无风险标签无描边
- [ ] 类别填充色沿用现有10类色板
- [ ] 可下钻标签右下角显示◎角标
- [ ] 三级节点尺寸比二级小，颜色继承所属类别

### 9.3 类别折叠/聚焦

- [ ] 点击类别名，该扇区放大占据60%圆
- [ ] 其他类别压缩为外圈小节点，仅显示类别名+标签总数
- [ ] 再次点击同一类别或点击中心头像，恢复全景
- [ ] 聚焦模式下可点击其他类别直接切换
- [ ] 聚焦模式下仍可点击二级标签进入三级展开

### 9.4 悬停高亮与 Tooltip

- [ ] 悬停二级节点，该节点放大1.2倍+所属类别高亮+连线加粗，其他淡化至30%
- [ ] 悬停三级节点，该节点放大1.2倍+所属二级高亮，其他三级淡化至30%
- [ ] 悬停类别名，该类别所有二级节点高亮，其他类别淡化至30%
- [ ] 悬停200ms后显示tooltip，内容包含标签名/风险等级/生成时间
- [ ] Tooltip位置在节点上方居中，不遮挡其他节点

### 9.5 搜索定位

- [ ] 搜索框位于图谱区右上角
- [ ] 输入关键词实时匹配二级和三级标签名（模糊匹配）
- [ ] 匹配结果>1时展示下拉列表（最多10条）
- [ ] 选中某条后图谱自动平移使节点居中+高亮闪烁2次
- [ ] 匹配结果=0时提示"未找到匹配标签"
- [ ] 搜索范围默认为当前筛选条件下可见标签

### 9.6 三级标签下钻

- [ ] 点击二级标签，该节点居中放大并向外辐射展开三级（180°扇形）
- [ ] 同时右侧自动展开溯源面板
- [ ] 单二级展开三级≤12个，超过时显示"+N"汇聚节点
- [ ] 点击三级标签，溯源面板内容更新为该三级证据
- [ ] 点击其他二级标签，直接切换三级展开（无需先回全景）
- [ ] 点击中心头像或Esc，收起三级回到全景
- [ ] 点击"+N"汇聚节点，展开剩余三级标签
- [ ] 展开动画300ms，节点依次出现（间隔50ms）

### 9.7 溯源信息面板

- [ ] 点击二级/三级标签时，右侧展开溯源面板（宽度320px）
- [ ] 面板包含基本信息区块：标签名/所属类别/风险等级/生成时间/数据截止
- [ ] 面板包含命中证据区块：首次出现/机构/科室/原文/关联事实数
- [ ] 点击新节点时面板内容更新（面板保持打开）
- [ ] 点击面板关闭按钮或Esc，面板收起
- [ ] 命中证据某字段无数据时显示"暂无数据"（灰色）
- [ ] 关联事实数为0时显示"该标签暂无关联事实记录"

### 9.8 筛选能力

- [ ] 顶部工具栏包含标签类别多选下拉和时间窗单选下拉
- [ ] 标签类别默认全选，时间窗默认"全部"
- [ ] 筛选变更后图谱实时刷新（无需点击查询按钮）
- [ ] 筛选结果为空时显示空态提示
- [ ] 页面刷新后筛选状态重置为默认值
- [ ] 图例改为右下角浮窗，默认折叠
- [ ] 图例展开后显示可展开/不可展开/可下钻说明+风险等级边框说明

### 9.9 合规标识

- [ ] 图谱区右上角固定展示"已脱敏展示"徽章
- [ ] 图谱区顶部展示"数据截止：YYYY-MM-DD"
- [ ] 数据截止时间与主界面患者头部一致

### 9.10 异常场景

- [ ] 患者无任何标签时，图谱区显示空态："该患者暂无健康标签数据"
- [ ] 筛选后无匹配标签时，显示："当前筛选条件无匹配标签，请调整筛选条件"
- [ ] 网络异常导致数据加载失败时，显示错误提示+重试按钮
- [ ] 溯源面板数据加载失败时，面板内显示错误提示+重试按钮
